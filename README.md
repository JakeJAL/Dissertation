# Data Science & Machine Learning Projects

A collection of Jupyter notebooks exploring various data science, machine learning, and simulation concepts.

## Project Structure

```
├── notebooks/          # Jupyter notebooks with analysis and simulations
├── data/              # Dataset files
└── models/            # Trained machine learning models
```

## Notebooks

### Machine Learning
- **intro_machine_learning.ipynb** - Introduction to machine learning concepts
- **video_game_sales_ml.ipynb** - Video game sales prediction using decision trees
- **music_recommender.dot** - Music recommendation system visualization

### Game Simulations & Analysis
- **overwatch_team_combat_simulation.ipynb** - Team combat mechanics simulation
- **overwatch_hero_stats.ipynb** - Hero statistics analysis
- **team_combat_mechanics.ipynb** - Combat mechanics modeling
- **game_simulation_scoring.ipynb** - Game scoring simulations
- **game_matchup_permutations.ipynb** - Matchup permutation analysis

### Probability & Statistics
- **sports_scoring_probability.ipynb** - Sports scoring probability calculations
- **scoring_probability_distribution.ipynb** - Probability distribution analysis

### Graph Theory & Networks
- **networkx_graph_visualization.ipynb** - Graph visualization using NetworkX
- **graph_adjacency_matrices.ipynb** - Graph adjacency matrix generation

## Datasets

- `music.csv` - Music dataset for recommendation system
- `vgsales.csv` - Video game sales data

## Models

- `music_recommender.joblib` - Trained music recommendation model

## Requirements

```
pandas
numpy
scikit-learn
networkx
matplotlib
joblib
```

## Installation

```bash
pip install pandas numpy scikit-learn networkx matplotlib joblib
```

## Usage

Launch Jupyter Notebook and open any notebook:

```bash
jupyter notebook
```
